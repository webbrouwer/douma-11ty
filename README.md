# Douma Dakdekkers

## Getting Started

### 1. Clone this Repository

```
git clone 'url'
```

### 2. Navigate to the directory

```
cd douma-11ty
```

### 3. Install dependencies

```
npm install
```

### 4. Build project

```
npm run build
```

### 5. Run Eleventy

```
npm run start
```
